Bem-vindo ao nosso projeto "Tropa de Elite"

Este projeto foi pedido pelo docente Eduardo Hernandes com o objetivo de exercitar os nossos conhecimentos acerca de html.

Atividade de criação de um projeto utilizando a linguagem de marcação HTML5.

Essa atividade, proposta pelo professor Eduardo consiste na criação de um projeto com 3 páginas contendo conteúdo. Estruturalmente esta página é dividida em: Header, Nav, Div, Section e Footer.
